# Grab ESG Report 2021

An interactive ESG microsite experience developed for :contentReference[oaicite:0]{index=0} to communicate the company’s 2021 Environmental, Social, and Governance initiatives through immersive digital storytelling.

Built with Vue 3, Vite, TailwindCSS, and GSAP to transform complex sustainability reporting into a responsive and engaging frontend experience.

## Overview

This project focuses on delivering long form ESG content through interactive storytelling, smooth animations, and responsive user experiences.

The microsite combines modular frontend architecture, scroll driven interactions, and animated visual transitions to improve readability, accessibility, and engagement across sustainability report sections.

---

## Features

- Responsive multi page frontend experience
- GSAP powered animations and transitions
- Scroll driven storytelling interactions
- Vue 3 component based architecture
- TailwindCSS utility first styling
- SVG icon system integration
- Social sharing support
- Embedded media and YouTube integrations
- Mobile optimised layouts
- Performance focused frontend implementation

## Local Development

### Recommended Environment

- Node.js 16+
- npm or yarn

---

### Install Dependencies

```bash
npm install
```

---

### Start Development Server

```bash
npm run dev
```

Runs locally at:

```bash
http://localhost:5173
```

---

### Production Build

```bash
npm run build
```

---

### Preview Production Build

```bash
npm run preview
```
